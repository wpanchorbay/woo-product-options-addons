Add custom product options, add-ons, and extra fields to WooCommerce products with advanced pricing, inventory, and conditional logic.

== Description ==

**OptionBay - Product Options and Addons** helps WooCommerce stores add flexible product fields, paid add-ons, swatches, conditional logic, and inventory-aware options to product pages.

**Key Features:**

*   Create text, textarea, number, email, file, select, checkbox, radio, color swatch, image swatch, and static content fields.
*   Configure flat, percentage, character-count, and formula pricing.
*   Assign option groups globally or to specific products, categories, and tags.
*   Track inventory at the field or option level.
*   Export and import option groups, inventory, and settings.

**Developer Friendly:**

*   React and TypeScript admin screens.
*   REST API powered option group and inventory management.
*   WooCommerce cart, checkout, and order meta integration.

== Installation ==

1.  Clone the repository or download the zip.
2.  Run `npm install` to install dependencies.
3.  Run `npm run build` to compile assets.
4.  Activate the plugin in WordPress.

== Frequently Asked Questions ==

= Does this require WooCommerce? =
Yes. WooCommerce must be active because options are rendered and priced on WooCommerce product pages.

= Can add-ons change the product price? =
Yes. Options can add flat fees, percentage-based fees, character-count pricing, or formula-based prices.
